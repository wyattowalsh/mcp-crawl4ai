## Community Contributions

Crawl4AI-MCP is an open-source project, and we welcome contributions from the community. Here are some ways you can contribute:

- **Feature Development**: Implement new features or enhance existing ones
- **Bug Fixes**: Help identify and fix bugs
- **Documentation**: Improve documentation, add examples, and clarify usage
- **Testing**: Write unit tests and integration tests
- **Feedback**: Provide feedback on features, usability, and performance

For more information on how to contribute, please see our [Contributing Guide](/docs/contributing).

## Feature Requests

We maintain a backlog of feature requests and ideas for future development. If you have a feature request, please open an issue on our [GitHub repository](https://github.com/wyattowalsh/crawl4ai-mcp/issues) with the "feature request" label.

<Callout type="info">
  Want to prioritize specific features or collaborate on development? Reach out to us on [GitHub](https://github.com/wyattowalsh/crawl4ai-mcp).
</Callout>
